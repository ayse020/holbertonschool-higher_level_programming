# SQL Introduction

This project contains SQL scripts for basic MySQL operations as part of the Holberton School curriculum.

## Tasks

0. **List databases** - Script that lists all databases of MySQL server
1. **Create a database** - Script that creates the database `hbtn_0c_0`
2. **Delete a database** - Script that deletes the database `hbtn_0c_0`
3. **List tables** - Script that lists all tables of a database
4. **First table** - Script that creates a table `first_table`
5. **Full description** - Script that prints the full description of `first_table`
6. **List all in table** - Script that lists all rows of `first_table`
7. **First add** - Script that inserts a new row in `first_table`
8. **Count 89** - Script that displays the number of records with `id = 89`
9. **Full creation** - Script that creates and fills a table `second_table`
10. **List by best** - Script that lists all records of `second_table` ordered by score
11. **Select the best** - Script that lists records with score >= 10
12. **Cheating is bad** - Script that updates Bob's score to 10
13. **Score too low** - Script that removes records with score <= 5
14. **Average** - Script that computes the average score
15. **Number by score** - Script that lists the number of records by score
16. **Say my name** - Script that lists all records with a name value

## Usage

Each script can be executed with:
```bash
mysql -u root -p < script_name.sql
