# SQL More Queries

This project contains SQL scripts for managing MySQL databases, users, and privileges.

## Project Structure

- `0-privileges.sql`: Lists all privileges of MySQL users user_0d_1 and user_0d_2

## Tasks

### Task 0: My privileges!
Write a script that lists all privileges of the MySQL users user_0d_1 and user_0d_2 on your server (in localhost).

## Requirements
- MySQL 8.0+
- Ubuntu 20.04 LTS

## Usage

To execute a SQL script:
```bash
mysql -hlocalhost -uroot -p < script_name.sql
