# Python Exceptions

This project contains Python scripts for exception handling exercises.

## Files
- 0-safe_print_list.py: Safe list printing function
- 1-safe_print_integer.py: Safe integer printing function
- 2-safe_print_list_integers.py: Print and count integers function
- 3-safe_print_division.py: Safe division function
- 4-list_division.py: Divide element by element function
