# Python Import Modules

This project is about learning how to import modules in Python.

## Task 0: Import a simple function from a simple file

This task requires importing the `add` function from `add_0.py` file and calculating 1 + 2.

### Requirements:
- Use only one occurrence of `add_0` in your code
- Do not use `*` for importing
- Code should not execute when imported
- Use only one print function with string formatting
