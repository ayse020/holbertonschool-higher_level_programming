# Python - Object-relational mapping

This project covers the basics of using Python to interact with relational databases using MySQL. 
It involves using the MySQLdb module to connect to a MySQL database and perform CRUD operations.

## Project Structure

- `0-select_states.py`: Lists all states from the database `hbtn_0e_0_usa`

## Requirements

- Python 3.x
- MySQLdb module (`pip install mysqlclient`)
- MySQL server

## Usage

### Task 0: Get all states
```bash
./0-select_states.py <mysql username> <mysql password> <database name>
